Facial Sketch Editing

Facial Sketch Expression Editing Project.

For criminal investigation and portrait drawing, automatic expression editing tools for face sketch improve work efficiency obviously and reduce professional requirements for users. In this project, we propose a novel method for facial expression editing in face sketch. The new facial expressions in the sketch images can be regenerated automatically.
